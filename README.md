# Python 练习项目

这个仓库用于记录我的 Python 编程练习，目标是通过小项目提升基础语法、文件处理、异常处理和命令行程序开发能力。

## 练习列表

### 练习 1：学员花名册管理器

文件：`roster_manager.py`

功能：
- 添加学员
- 查看学员列表
- 管理简单的学员信息

运行：

```bash
python3 roster_manager.py
```

### 练习 2：文本词频统计器

文件：`word_counter.py`

功能：
- 读取文本
- 统计单词出现次数
- 输出词频结果

运行：

```bash
python3 word_counter.py
```

### 练习 3：待办事项清单

文件：`todo_list.py`

功能：
- 添加待办事项
- 完成待办事项
- 查看待办清单
- 使用 `todos.json` 保存数据
- 程序重启后可以加载之前的数据

运行：

```bash
python3 todo_list.py
```

### 练习 4：安全的计算器

文件：`safe_calculator.py`

功能：
- 支持加、减、乘、除
- 支持 `*`、`×`、`/`、`÷`
- 除数为 0 时不会崩溃
- 输入非数字时不会崩溃
- 输入 `quit` 可以退出程序

运行：

```bash
python3 safe_calculator.py
```

## 学习重点

通过这些练习，我学习了：

- Python 基础输入输出
- 条件判断和循环
- 列表、字典等基础数据结构
- 函数拆分
- JSON 文件读写
- 异常处理
